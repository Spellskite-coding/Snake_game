Bonjour !

Pour jouer au jeu du snake sur votre ordinateur via le script, il vous suffit d'avoir python d'installé sur votre machine !

Ensuite, en fonction de votre OS, vous devrez installer la bibliothèque python nommée "pygame" !

Ensuite il vous suffit de lancer le jeu via la commande "python3 snake.py", mais biensur cela dépends aussi de votre OS !

Amusez-vous bien avec le jeu du snake !
